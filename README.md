# Mastermind
Mastermind is a CLI game written in C#.

## Requirements
* .NET Core SDK v3.1 [download](https://dotnet.microsoft.com/download)

## Installation
```$ git clone https://github.com/inspiredtolive/mastermind.git```

## Usage
```
$ cd mastermind
$ dotnet run
```
